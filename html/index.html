Proyectos

E-Commerce Mercado Liebre

En este proyecto, desarrollé una interfaz de usuario para una plataforma de e-commerce "Mercado Liebre". El diseño incluye vistas adaptadas a dispositivos mobile, tablet y desktop asegurando una experiencia de usuario óptima.

La interfaz ofrece funcionalidades como recomendaciones personalizadas basadas en las últimas visitas del usuario, ofertas de productos con descuentos y diferentes opciones de pago. La estructura visual es clara y moderna, enfocada en la experiencia del usuario.

E-Commerce Kitchening

El proyecto Kitchening fue desarrollado en colaboración con mis compañeros y profesor de cursada, con el objetivo de crear un sitio web funcional y practicar lo visto en clase.

Nos enfocamos en diseñar una plataforma que fuera atractiva y fácil de navegar, con énfasis en la presentación visual de los platos y servicios ofrecidos por el restaurante.

E-Commerce Home Deco

Este proyecto consistió en el desarrollo completo de una página web para Home Deco (ficticia), enfocada en ofrecer una experiencia de usuario intuitiva y visualmente atractiva.

El sitio web presenta un catálogo de productos cuidadosamente organizado, destacando piezas clave como sofás, sillas, y mesas, con funcionalidades modernas.
